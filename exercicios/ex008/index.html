Principais Formatação

Negrito / Destaque

Nessa frase temos termo em negrito usando a tag B (não semântica).

Nessa frase temos um termo em destaque usando a tag STRONG (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântica).

Nesta frase temos um termo em ênfase usando a tag EM (semântica).

Texto marcado

Podemos criar tamném um texto marcado usando a tag MARK.

podemos criar também um testo marcado usando a tag MARK e STYLE.

Testo grande e pequeno

Estamos crindo um texto grande e um texto pequeno nesse parágrafo. usando:
BIG (Não semântica)
SMALL (semântica).

Texto deletado

podemos marcar um texto como excluido para undicar que ele deve ser lido, mas não considerado. usando:
DELL (semântica).

Texto inserido

Podemos marcar um texto como inserido para dar uma infase e indicar que ele foi adicionado depois. usando:
INS (semântica).

Existe também o sublinhado com a tag U (não semântica)

Texto sobrescrito

Para inserir coisa do tipo x20+3. usando a tag SUP.

Texto subscrito

Para inserircoisa do tipo H2O. usando a tag SUB.
